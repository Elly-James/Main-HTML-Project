# My Leisure Life Website

[Preview Link](https://elly-james.github.io/Main-HTML-Project/)
## Overview

This is a simple HTML project that showcases how I spend my free time. The website includes different pages that highlight my hobbies, favorite movies, books I enjoy reading, and my favorite soccer team.

## Pages Included

* **Home (Index Page):** Introduction to the website and a suggestion form for feedback.
* **Hobbies:** Shows activities I enjoy such as swimming and listening to music.
* **Favorite Movies:** Displays my top movies with images and links to watch their trailers on YouTube.
* **Favorite Books:** Lists books I like with short descriptions and links to read more.
* **Soccer Team:** Highlights my favorite team (Manchester United) and includes a table rating my top players.

## Features

* Simple and clean HTML structure
* Use of semantic HTML elements like header, main, section, and footer
* Navigation between pages
* Images and external links
* Embedded media (where possible)
* A suggestion form for user feedback

## Technologies Used

* HTML5

## Purpose

The purpose of this project is to practice basic web development skills, including structuring web pages, linking multiple pages, and using different HTML elements.
